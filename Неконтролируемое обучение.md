Суть: данные, в отличии от [[контролируемое обучение|контролируемого обучения]] без таргета. То есть машина ищет совпадения или закономерности "вслепую"
![[Screenshot 2025-11-21 at 10.03.18.png]]
Применяется для неразмеченных данных и состоит из алгоритмов, упрощающих данные без потери большого количества информации.

**Два распространенных типа алгоритмов**:

1. **Кластеризация** — группировка похожих данных в кластеры для извлечения информации
    
2. **Понижение размерности** — способ упростить данные, объединив схожие признаки
