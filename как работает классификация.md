**Основной принцип**: Поиск разделяющей границы между группами объектов с разными метками.

**Математическая формулировка**:

- Строится функция, которая разделяет пространство признаков на области
    
- **Решающее правило**: Если точка попадает в область A → класс A, если в область B → класс B
    
- Линейный классификатор: $f(x) = w_1x_1 + w_2x_2 + \dots + b$
    
- Если $f(x) > 0$ → Класс 1, иначе → Класс 0
    

**Пример применения** (фильтр спама):

- **Признаки**: частота слов "бесплатно", "выигрыш", наличие ссылок, восклицательные знаки, имя отправителя
    
- **Метка**: Спам / Не спам
    
- **Веса**: определяют "подозрительность" каждого слова (слово "выигрыш" имеет большой вес для класса Спам)
    

**Алгоритм обучения**:

1. Получаем объекты с известными правильными ответами
    
2. Ищем границу (линию или гиперплоскость), которая лучше всего отделяет "крестики" от "ноликов"
    
3. Штрафуем модель за каждый объект, оказавшийся по "чужую" сторону границы
    
4. Сдвигаем границу, чтобы минимизировать ошибки
