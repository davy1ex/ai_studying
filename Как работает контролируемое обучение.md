**Суть**: Модель "учится" на примерах, где известен правильный ответ, чтобы затем предсказывать ответы для новых примеров.

**Определение** Используется для размеченных данных и заключается в построении моделей, предсказывающих метки для новых данных.

**Два распространенных типа алгоритмов**:

1. **[[как работает линейная регрессия|Регрессия]]** — ответ служит любое число
    
2. **[[как работает классификация|Классификация]]** — ответ относится к типу или классу
