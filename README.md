# Что это?
Это страница индекс. Она начало в данный мир [[искусственный интеллект|Искуственного интеллекта]]. 

# Как этим пользоваться?
1. скачать с помощью кнопки Code -> download ZIP
2. расспаковать
3. скачать Obsidian как удобную читалку данных заметок https://obsidian.md
4. открыть там Vault как расспакованный данный проект
5. (опционально) разрешить доверие плагинам (так удобнее пользоваться, т.к. я преднастроил конфиг читалки Obsidian для вашего удобства)
6. открыть [[Регрессия.canvas]]
