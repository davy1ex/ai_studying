Существует несколько типов МО, которые выбираются в зависимости от структуры данных:

1. **[[Как работает контролируемое обучение|Контролируемое обучение]]** — для размеченных данных
    
2. **[[Неконтролируемое обучение]]** — для неразмеченных данных
    
3. **Обучение с подкреплением** — для взаимодействия с окружением

**Критическое различие**: Данные могут быть размеченными (содержат метку (или ещё её называют таргет) для предсказания) или неразмеченными (метки отсутствуют).
